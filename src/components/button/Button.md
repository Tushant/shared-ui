1. Solid Buttons
```jsx padded
<Button variant="solid" colorVariant="accent">Accent Solid Button</Button>
<Button variant="solid" colorVariant="danger">Danger Solid Button</Button>
<Button variant="solid" colorVariant="neutral">Neutral Solid Button</Button>
<Button variant="solid" isDisabled>Disabled Solid Button</Button>
<Button variant="solid" isLoading>Loading Solid Button</Button>
<Button variant="solid" isBlock>Block Solid Button</Button>
<Button variant="solid" size="sm">Small Solid Button</Button>
<Button variant="solid" size="lg">Large Solid Button</Button>
<Button variant="solid" shape="pill">Pill Solid Button</Button>
```
